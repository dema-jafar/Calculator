Keyboard Support

The calculator supports the following keyboard keys:

| Key              | Function            |
|------------------|---------------------|
| Numbers          | Enter digits        |
| `+ - * / . ( )`  | Arithmetic operators|
| `Enter`          | Calculate (=)       |
| `Backspace`      | Delete last input   |
| `Escape`         | Clear all           |
